# Herbs Store

Moderne, luxe en responsive fashion e-commerce homepage gebouwd met React, TypeScript, Tailwind CSS, React Router en Framer Motion.

## Starten

```bash
npm install
npm run dev
```

## Features

- Dynamische homepage content via mock data
- Announcement bar
- Header met mega menu en mobile menu
- Hero carousel
- Category grid
- Promo blocks
- Product grid met filter en search
- Editorial/story section
- Newsletter CTA
- Grote footer
- Responsive design
- Subtiele animaties
